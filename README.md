# Scramble-Inverter
Invert a Rubik's Cube scramble or move sequence in your clipboard. Can also generate more text for starting your attempt.

# Setup
## Windows Setup
Navigate to the newest release and download the Scramble Inverter Setup file and follow the instructions.
## Python Setup
Download the file ScrambleInverter.py. Make sure you have the library pyperclip installed. Navigate to the directory to which you have downloaded the file in your terminal of choice and run "python ScrambleInverter.py".

# How to use
Copy the scramble or move sequence you want to invert, either with ctrl+c (cmd+c on Mac) or by clicking the scramble, if it's in CSTimer. With the scramble in your clipboard, launch the program. This will open a terminal window and put the inverted scramble in your clipboard.

- Press x, then enter, to recover the original scramble into your clipboard
- Press spacebar, then x, to copy a longer piece of text with the format:
  Scramble: *scramble*
  
  Inverse: *Inverse*
- Press Enter to close the program and keep just the inverse scramble in your clipboarc
